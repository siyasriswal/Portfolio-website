# 🌐 AI/ML Portfolio Website

A modern, responsive portfolio website showcasing my journey in **Artificial Intelligence, Machine Learning, and Data Science**. The website highlights my technical skills, projects, certifications, education, and provides an interactive experience through animations, project filtering, and a live Fake News Classifier demo.

## ✨ Features

* 📱 Fully Responsive Design
* 🎨 Modern Dark UI
* ⚡ Smooth Scroll & Reveal Animations
* 🤖 Interactive Fake News Classifier Demo
* 📚 Education Timeline
* 💻 Technical Skills Showcase
* 📂 Project Filtering by Category
* 📖 Expandable "How It Works" Pipelines
* 🏆 Certifications Section
* 📬 Contact Section
* 🌌 Animated Background Effects

## 🛠️ Built With

* HTML5

* JavaScript (ES6)
